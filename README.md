# 🌅 Ruang Nostalgia

> Blog analisis fan-made tentang *A Space for the Unbound* — menyelami narasi psikologis, estetika pixel art Jawa Timur era 90-an, dan melodi nostalgia dari mahakarya Mojiken Studio.

---

## ✨ Tentang Proyek

**Ruang Nostalgia** adalah sebuah blog statis yang dibuat sebagai proyek portofolio pribadi. Blog ini mengeksplorasi berbagai aspek mendalam dari game *A Space for the Unbound*, mulai dari desain mekanika, analisis psikologi karakter, worldbuilding Kota Loka, hingga kajian musik dan visual pixel art-nya.

Website ini dibangun dengan **HTML, CSS, dan JavaScript murni** tanpa framework — mengedepankan estetika senja yang hangat, tipografi yang nyaman dibaca, dan pengalaman membaca yang imersif.

## 🚀 Cara Menjalankan

Tidak ada dependency atau build step. Cukup buka langsung di browser:

```bash
# Clone repository
git clone https://github.com/irvy002/RuangNostalgia01.git

# Buka di browser
cd RuangNostalgia01
# Klik dua kali index.html, atau gunakan Live Server di VS Code
```

Atau kunjungi versi live di **GitHub Pages** *(tambahkan link setelah deploy)*.

## 🎨 Fitur

- **Desain responsif** — optimal di desktop, tablet, dan mobile
- **Page transition** animasi *Space Dive* & *Flashback* saat berpindah halaman
- **Modal karakter** interaktif di halaman Kisah & Karakter
- **Easter egg** kucing Admiral yang bisa diklik di setiap halaman
- **Particle cursor** efek bintang berjatuhan (desktop only)
- **Back-to-top** button dengan smooth scroll
